## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #8b5cf6;
    --secondary-color: #6366f1;
    --dark-bg: #0a0a0f;
    --card-bg: rgba(17, 17, 27, 0.8);
}
```

### Particle Effect
Modify particle settings in `script.js`:
```javascript
particlesJS('particles-js', {
    particles: {
        number: { value: 80 },
        color: { value: '#8b5cf6' },
        // ... more settings
    }
});
```
